## Installation:-
### Steps
*  fork the repository
* `git clone <repository-url>` this repository.
* `cd T-Picker-Backend`

### Install dependencies:-
```
npm install
```
### Run The Development Server:-

Running:
-   `npm start`
-   Visit your app in local system [http://localhost:8000](http://localhost:8000).
